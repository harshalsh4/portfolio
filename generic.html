1. React App

SmartBrainy

It is a face recognition app. The frontend is created with HTML, CSS, JavaScript and React. A server with Node and Express, and a secured data base using PostgreSQl.
How to use:
1. Press the Live Demo button below
2. Press Test button on top right corner of the website
3. Paste an image-link in the empty box and click Detect.
The portion of the image where the face is visible will be highlighted by a blue box.

3. Natural Language Processing

Literature Skimit

In this project, we're going to be replicating the deep learning model behind the 2017 paper PubMed 200k RCT: a Dataset for Sequenctial Sentence Classification in Medical Abstracts. When it was released, the paper presented a new dataset called PubMed 200k RCT which consists of ~200,000 labelled Randomized Controlled Trial abstracts (Part of TensorFlow certification course). The goal of the dataset was to explore the ability for NLP models to classify sentences which appear in sequential order.
